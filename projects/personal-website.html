Personal Website

Originally a personal project, the aim of this website was to provide a couple of basic pages about myself, however as I continued to produce other projects, I decided to repurpose and expand the website to act as a central hub.

Along side external projects that I've made, the table below features the utilities hosted on this website that I created for personal use and have grown into something that others may find a use for.
These utilities are slowly being converted from python into javascript so that they can be run locally in the browser, meaning there will be no need for any installation and nothing will be sent to a server.
No personal data is collected when you use the utilities on this website, I dislike it, there's a good chance you dislike it too, and it's a lot more effort on my part to collect it.

Colour Palette Replacer (WIP) Replaces specific colours on an image based on a user provided palette
Content Pack Creator (Depreciated) Creates the base files for various content packs (Use Stardew Valley: Deploy and Pack VSCode extension instead)
